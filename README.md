# でかマウス (DecaMouse)

## 概要
マウスポインターを見失った際に、横または縦に振ることで見つけやすくなるかもしれないユーティリティです。  
横、または縦に振った際に以下の動作を割り当てられます。
* マウスポインターを大きくする
* マウスポインターをプライマリーディスプレイの中央に移動する
  
既定では横に振るとポインターを拡大し、縦に振るとポインターを中央に移動します。  

## インストール
インストーラーに従いインストールしてください。  
> ⚠️ ユーザー別インストール推奨  
> * 共有インストールで複数ユーザーが自動起動を設定した場合は、アンインストーラーではアンインストールを行ったユーザーのみしか解除できません。  
> * 共有インストールで複数ユーザーが自動起動を設定している場合は、前もって自動起動を解除してからアンインストールしてください。

## 使い方
既定では以下の動作が設定されています。  
* 横にマウスを振るとマウスポインターが大きくなります。振るのをやめると元に戻ります。
  * アクセシビリティ > マウス ポインターとタッチのサイズを操作しています。  
  既に最大サイズの場合は動作しません。

* 縦に振るとマウスポインターをディスプレイ中央に移動します。マルチディスプレイ環境ではプライマリディスプレイの中央です。
* タスクトレイ通知領域のアイコンから設定変更および終了できます。

## 設定について
タスクトレイ通知領域のアイコンから設定を選ぶか、アイコンをダブルクリックすると設定画面が開きます。  
横に振ったとき、縦に振ったときの動作と振りの感度調整が出来ます。  
感度調整は左下部の振りチェックのカウンターで確認しながら調整してください。  
マウスポインターの大きくなる速度や、Windows起動時の自動起動設定も行えます。
